### 简介

使用 react 完成后台管理平台开发。

服务后台使用 nestjs + mysql + redis 开发。

### 启动

```shell
yarn

yarn start
```

### 进度

1. 登录
2. 用户管理
   1. 列表显示
   2. 添加
   3. 更新
   4. 删除
3. 权限管理
   1. 列表显示
   2. 添加
   3. 修改
4. 账号管理
   1. 添加
   2. 修改
   3. 搜索
   4. 批量删除
5. 币种管理
   1. 列表显示
   2. 添加
   3. 更新
   4. 根据 ID 获取分类名称
6. 主页信息显示
   1. 账号统计
   2. 币种统计
   3. 余额监控
